Supported Datastores

Metamapper currently supports automatic crawling and indexing of six SQL-based relational database management systems with plans to add more in the near future.

  • Amazon Redshift
  • Microsoft SQL Server
  • MySQL
  • Oracle
  • PostgreSQL
  • Snowflake
